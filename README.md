Tento repozitár obsahuje zdrojové kódy k bakalárskej a diplomovej práci zameraným na oblasť vysvetliteľnosti modelov umelej inteligencie. Každý priečinok (`BP`, `DP`) obsahuje vlastný `README.md` súbor, kde sú uvedené podrobnejšie informácie.

## Štruktúra repozitára

Repozitár je rozdelený do dvoch hlavných priečinkov:

### 1) BP
Obsahuje kód k bakalárskej práci: **"Porovnanie rôznych metód vysvetliteľnosti modelov umelej inteligencie na zvolenej množine dát"**


### 2) DP
Obsahuje kód k diplomovej práci: **"Metódy hodnotenia kvality vysvetliteľných modelov umelej inteligencie"**
